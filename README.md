# 📝 AdvanceTodo App

A high-performance, full-stack Task Management application. This project features a modern Kanban-style interface, dynamic drag-and-drop functionality, and a robust **Soft-Delete (Trash)** system for data safety.

---

## 🚀 Key Features

* **Kanban Board:** Organize tasks by status (To-Do, In Progress, Done) with a clean, responsive UI.
* **Drag & Drop:** Intuitively reorder tasks or move them between categories.
* **Soft Delete (Trash):** Tasks aren't lost immediately. Access the **Trash** section to restore them or purge them permanently.
* **Category Management:** Assign custom colors and names to categories for better visual organization.
* **Mobile Optimized:** Fully responsive design that scales from desktop monitors to mobile devices.
* **RESTful Architecture:** Built with Spring Boot for high-speed data processing and JPA for persistent storage.

---

## 🛠️ Tech Stack

### **Frontend**
* **React.js** (Hooks & Functional Components)
* **Tailwind CSS** (Modern Utility-first Styling)
* **Axios** (Backend API Integration)
* **Vite** (Lightning-fast Build Tool)

### **Backend**
* **Java 17+**
* **Spring Boot** (Web, Data JPA)
* **Database:** PostgreSQL / MySQL
* **Lombok:** For cleaner, boilerplate-free code
